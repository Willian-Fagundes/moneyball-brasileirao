# Moneyball Campeonato Brasileiro Serie A
# Projeto de análise de dados que investiga se gastar mais em contratações realmente melhora o desempenho dos clubes no futebol.

## Problema: 

O futebol moderno tem se tornado cada vez mais influenciado por fatores financeiros, especialmente no que diz respeito à contratação de jogadores. Clubes investem milhões em transferências com a expectativa de melhorar seu desempenho esportivo, mas nem sempre esse investimento se traduz em resultados dentro de campo.
Diante desse cenário, surge a seguinte questão central:
Existe uma relação direta entre o valor investido em contratações e o desempenho esportivo de um clube ao longo de uma temporada?

Este projeto tem como objetivo investigar essa relação utilizando dados de ligas como o Brasileirão, analisando se maiores investimentos resultam, de fato, em melhores campanhas.

## Objetivo

Analisar a correlação entre o investimento financeiro em transferências de jogadores e o desempenho esportivo dos clubes, medido por indicadores como pontuação, gols marcados/sofridos e posição final na tabela.

## Escopo do Projeto

### O projeto se concentra nas seguintes etapas:

### 1- Coleta de Dados:
   Dados de transferências (ex: valores, jogadores, clubes), 
   Dados de desempenho (tabela final, estatísticas da temporada)
### 2- Tratamento e Limpeza:
  Padronização de nomes de clubes:
  Conversão de moedas,
  Tratamento de valores ausentes
### 3- Análise Exploratória (EDA):
  Distribuição de gastos,
  Comparação entre clubes,
  Identificação de outliers (times que gastam muito e performam mal, ou vice-versa),
### 4- Análise de Correlação:
  Relação entre investimento e desempenho,
  Uso de correlação (Pearson/Spearman),
### 5- Visualização de Dados:
  Gráficos de dispersão (gasto vs pontos),
  Rankings comparativos,
  Dashboards interativos
